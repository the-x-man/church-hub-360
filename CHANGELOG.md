# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2025-11-28

### New Features
- **Finance Record Edit Locking & Request System**:
  - Implemented a "Lock by Default" mechanism for finance records (Income, Expenses) to ensure data integrity.
  - Added a request/approval workflow for editing finance records.
  - Introduced Edit Requests and Notifications tables to track access requests and approvals.
  - Owners can now review and approve/reject edit requests via a new notification system.

### Bug Fixes
- Fixed an issue where the auto-update button on Windows incorrectly displayed "Download from Website" instead of "Install Updates" when an update was ready.
